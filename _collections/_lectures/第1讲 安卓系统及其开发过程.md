**Android Studio**是由Google打造和发布的用于开发Android应用的官方集成开发环境。软件开发者可利用IDE中的工具为Android平台**设计**、**构建**、**运行**和**测试**软件。Android Studio以INtelliJ IDEA为基础，预装了Android插件，并且专门针对Android平台进行了一些修改。

本课程使用**Android Studio Hedgehog 2023.1.1 Patch 1**版本，对应的下载链接：

- [Windows](https://edgedl.me.gvt1.com/edgedl/android/studio/ide-zips/2023.1.1.27/android-studio-2023.1.1.27-windows.zip)

## 下载并安装Android Studio

### Android Studio系统要求

- 64 位 Microsoft® Windows® 8/10/11
- x86_64 CPU 架构；第 2 代 Intel Core 或更高版本，或者支持 [Windows Hypervisor](https://developer.android.google.cn/studio/run/emulator-acceleration?hl=zh-cn#vm-windows) 的 AMD CPU
- 至少有 8 GB 的 RAM
- 至少 8 GB 可用磁盘空间（IDE + Android SDK + Android 模拟器）
- 最低屏幕分辨率：1280 x 800

具体安装过程，请参考[设置Android Studio](https://developer.android.google.cn/courses/pathways/android-basics-compose-unit-1-pathway-2?hl=zh-cn)页面。
